# FinTech Mobile Application Wireframe


## Project Overview

This project focuses on designing a low-fidelity wireframe for a FinTech mobile application.
The wireframe demonstrates user flow, screen structure, navigation patterns, and financial management features before the final UI design stage.
The application allows users to manage accounts, transfer money, view transactions, track expenses, pay bills, and manage profile settings.

UI design and future development.
